## рџЋЇ Research Objective

This research proposes a lightweight Stateful IDS for IoT networks capable of reducing false positives through temporal decision-making.

Unlike conventional stateless IDS, every prediction generated by the machine learning model is first validated using a Finite State Machine (FSM) before producing the final detection status.

## рџ“Ў Network Telemetry Features

The IDS uses lightweight application-level telemetry collected directly from ESP32:

- Latency
- Jitter
- Interval
- Packet Rate

These features are used to classify network traffic as either:

- Normal
- DoS

---

## рџ¤– Machine Learning

Two supervised learning algorithms are used:

- Random Forest
- XGBoost

Both models are trained using telemetry datasets collected directly from ESP32 under normal and DoS attack scenarios.

---

## рџ›ЎпёЏ Stateful Detection States

Temporary Status

- NORMAL
- SUSPICIOUS
- DOS INDICATED

Final Status

- NORMAL
- UNDER MONITORING
- CONFIRMED DOS ATTACK
- DOS ATTACK IN PROGRESS
- RECOVERY MONITORING

The FSM validates predictions using consecutive anomaly counters before changing the final detection state.
